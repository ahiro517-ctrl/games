# 🧩 毎日パズル

ブラウザで手軽に遊べるパズルゲーム集です。ホーム画面から好きなゲームを選んで遊べます。

## 遊べるゲーム

| ゲーム | 内容 |
|--------|------|
| 🌻 ジップ | 1から順に数字をつないで、すべてのマスを通る一筆書きパズル |
| ⭐ サムズ | 数字を消して、残りの合計を行・列の目標に合わせるパズル |
| 🌸 ミニ数独 | 6×6のやさしい数独 |
| 🐼 パンダ | 🐼と🎍を各行・各列に3つずつ、3連続にならないように並べるパズル |
| 💡 ライツ | タップで十字にライトが反転。全部消したらクリア |
| 🔢 2048 | スワイプで同じ数字を合体させる定番パズル（フリープレイ） |

- ジップ〜ライツは**日替わり問題**です。日付から問題を生成するので、同じ日はみんな同じ問題になります。
- クリアするとカードに ✓ が付き、連続プレイ日数 🔥 がカウントされます（ブラウザに保存）。
- スマホでもPCでも遊べます。

## GitHub Pages で公開する方法

1. このブランチを `main` にマージする
2. リポジトリの **Settings → Pages** を開く
3. **Source** で「**GitHub Actions**」を選ぶ
4. `main` にプッシュすると自動でデプロイされ、`https://<ユーザー名>.github.io/games/` で遊べるようになります

（ワークフロー `.github/workflows/pages.yml` が自動デプロイを行います）

## ローカルで遊ぶ

ビルド不要の静的サイトです。`index.html` をブラウザで開くだけでも遊べます。

```bash
# 簡易サーバーで開く場合
python3 -m http.server 8000
# → http://localhost:8000
```
